# 보안 및 개인 정보 보호 정책

## 개인 정보 보호 우선 원칙
NUNCHI는 최소한의 데이터 보존 원칙에 따라 구축되었습니다. 이 서비스는 민감한 대화 데이터를 영구적으로 저장하지 않고 코칭 통찰력을 제공하도록 설계되었습니다.

## 제로 보존 프로토콜
제로 보존 프로토콜은 다음을 보장합니다:
- 원시 오디오 파일은 처리 중에 휘발성 메모리 또는 임시 암호화된 저장소에 저장됩니다.
- 모든 원시 데이터는 분석 완료 또는 사용자 확인 즉시 영구적으로 삭제됩니다.
- 명시적이고 확인 가능한 사용자 동의 없이는 어떠한 대화 데이터도 모델 학습에 사용되지 않습니다.

## 데이터 암호화
- 전송 중: 클라이언트와 서버 간에 전송되는 모든 데이터는 TLS 1.3 이상을 사용하여 암호화됩니다.
- 저장 중: 시스템에 저장된 모든 임시 데이터는 AES-256을 사용하여 암호화됩니다.

## 인프라 보안
당사의 인프라는 일관된 보안 구성을 보장하고 수동 오류를 최소화하기 위해 자동화된 오케스트레이션 스크립트(IaC)를 통해 관리됩니다. 프로덕션 환경에 대한 액세스는 다단계 인증을 통해 승인된 직원으로 엄격하게 제한됩니다.

## 취약점 공개
당사 플랫폼에서 잠재적인 보안 취약점을 발견하면 공식 보안 채널을 통해 보고해 주십시오. 당사는 보안 문제를 신속하게 처리하고 해결하기 위한 협력 공개 프로그램을 운영합니다.
