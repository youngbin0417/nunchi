# NUNCHI 공개 릴리스

이 저장소는 AI 기반 관계 코칭 서비스인 NUNCHI의 공식 공개 문서를 포함합니다.

---

## 시스템 아키텍처 개요

NUNCHI는 복잡한 AI 계산 작업을 효율적으로 처리하기 위한 분산 비동기 시스템으로 설계되었습니다:

- 프론트엔드: 사용자 상호 작용을 위한 Dart 기반 인터페이스.
- API 서버: FastAPI 기반 조정 계층.
- AI 워커: 추론 작업을 위한 분산 Celery 워커.
- 메시지 브로커: 작업 스케줄링 및 상태 관리를 위한 Redis.
- 공유 저장소: 안전하고 S3 호환되는 임시 저장소.
